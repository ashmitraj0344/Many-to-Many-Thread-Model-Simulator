# Many-to-Many-Thread-Model-Simulator
# Overview

This project is an interactive Many-to-Many (M:N) Thread Model Simulator built to demonstrate how modern operating systems efficiently manage concurrency.
It visualizes how:

Multiple User Threads (M)
Are mapped to multiple Kernel Threads (N)
Which are executed on CPU cores

The simulator provides a real-time UI + backend simulation engine, making it ideal for:

📚 Operating System learning
🎓 College projects & viva
🧠 Understanding thread scheduling and concurrency

# Core Concepts Demonstrated
-Many-to-Many Thread Model (M:N)
-User-Level Threads vs Kernel-Level Threads
-CPU Scheduling
-Context Switching
-Thread States (Ready, Running, Blocked, Completed)
-Concurrency & Parallelism
